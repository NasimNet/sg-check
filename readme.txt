== Description ==

By this plugin you can determine the Source Guardian is installed on the server or not

*   determine the Source Guardian is installed
*   Showing the version of the Source Guardian

= Persian Description =
با نصب و فعال سازی این افزونه می توانید متوجه شوید که سورس گاردین بر روی سرور شما نصب شده است یا خیر

* قابلیت نمایش نصب و یا عدم نصب سورس گاردین بر روی سرور
* نمایش نسخه سورس گاردین نصب شده بر روی سرور
* مشاهده اطلاعات سرور

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Tools -> SG Check screen to configure the plugin
