With this plugin you can check the version of your server's Source Guardian loader.

== Description ==

By this plugin you can determine the Source Guardian is installed on the server or not

*   determine the Source Guardian is installed
*   Showing the version of the Source Guardian

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Tools -> SG Check screen to configure the plugin
